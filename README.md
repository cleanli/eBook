# eBook
因为个人兴趣开发的一款界面简洁、风格清新、功能丰富的Android单本电子书应用：     
具有仿真翻页、覆盖翻页、直接翻页三种翻页风格;    
常规、复古、护眼、夜间四种阅读模式，可调节亮度，能根据光线自动调节亮度；           
支持文字大小、字体、颜色设置；
能自动保存阅读进度，支持目录和书签功能。
